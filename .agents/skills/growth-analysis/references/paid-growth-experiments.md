# Paid Growth Experiments

Use when planning or evaluating paid acquisition tests.

## Product-Stage Rule

Do not default to A/B tests for low-volume products.

| Stage | Better validation |
| --- | --- |
| Low traffic | directional campaign tests, qualitative lead review, sales/support call review, landing page or fake-door tests |
| Some traffic | campaign/cohort comparison, small budget tests, before/after reads with caveats |
| Meaningful traffic | A/B tests, holdouts, incrementality tests, budget ramp tests |
| Mature paid engine | geo holdouts, lift studies, MMM/MTA, portfolio optimization |

Use formal experiments only when sample size, event frequency, attribution quality, and risk controls are sufficient.

## Test Types

- creative test
- audience test
- keyword/query test
- landing page test
- offer/pricing test
- channel test
- budget incrementality test
- geo holdout
